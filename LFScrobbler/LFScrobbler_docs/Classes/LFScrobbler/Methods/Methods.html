endSession

Indicate that a session has ended. 
- ( void ) endSession;

This method performs a "stop" and sends any un-Scrobbled changes to Audioscrobbler. Note that this method uses a blocking request, rather than an asynchronous one. This can present problems, but also allows you to use it in an applicationWillQuit: environment.

loginWithUsername:password:

Login to the Last.fm service. 
- ( void ) loginWithUsername:
        (NSString *) username password:
        (NSString *) password;

This method performs a handshake with the Scrobbler service to begin communication. It is recommended that you store the username and password in the Keychain, rather than in cleartext.

Parameter Descriptions
username
Username of the application's user on Last.fm.
password
Password of the application's user on Last.fm.

pause

Tell LFScrobbler that the user has paused the client. 
- ( void ) pause;

This method stops the play counter temporarily, which is important for sending accurate play statisticts to Last.fm

playTrack:

Tell LFScrobbler that a new track has started playing. 
- ( void ) playTrack:
        (NSDictionary *) track;

It is unnecessary to first send a stop command to LFScrobbler: calling playTrack: automatically does this.

Parameter Descriptions
track
Information on the track that has just started playing. The dictionary should contain the following keys: name (NSString): track's name; artist (NSString): track's artist; album (NSString): track's album (empty string if not known); mbID (NSString): MusicBrainz ID (optional); length (NSNumber): track's length in seconds; number (NSNumber): track's position on album (optional);

resume

Tell LFScrobbler that the user has resumed playing the track. 
- ( void ) resume;

This resumes the play counter where it left off.

setClientID:version:

Set the client ID of your software for use with Audioscrobbler. 
- ( void ) setClientID:
        (NSString *) newID version:
        (NSString *) newVersion;

All clients interfacing with Last.fm are required to have their own client ID. Obtain one for your client by contacting russ@last.fm, then assign it with this method before logging in. You should also provide the version of your application. If a client ID is not set, the testing ID will be used.

Parameter Descriptions
newID
The client ID assigned to you by Last.fm.
newVersion
The version of your client.

setDelegate:

Set the delegate of the LFScrobbler object. 
- ( void ) setDelegate:
        (id ) newDelegate;
Parameter Descriptions
newDelegate
An object to act as a delegate of the LFScrobbler object, implementing any assortment of methods from LFScrobblerDelegate.

stop

Tell LFScrobbler that the current track has stopped playing. 
- ( void ) stop;

Sending this will Scrobble the track, if the previous track has been playing for more than 320 seconds. Alternatively, you can simply send a playTrack: command if a new track has started playing (you do not need to send both commands).
